01_01_Python DA Project ReadMe
Github Link: https://github.com/Bobby-Mack1/Crime_DA_Project.git
This project showcases the use of python to clean, analyse and visualise data. The data used is from: Data downloads | data.police.uk, with the project focusing solely on crime in Wales. The purpose for analysis is for research for potential investments of a real estate company.

For Data go to https://data.police.uk/data/ and select jan 2023 - Dec 2024. Then select the following 4 police forces: Dyfed-Powys, North and South Wales and Gwent. Download and then rearrange the data month folders into police force folders.
